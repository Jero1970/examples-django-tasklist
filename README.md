# MemCachier and Django on PythonAnywhere tutorial

This is an example Django 2.1 task list app that
uses a [MemCachier](https://www.memcachier.com/) cache on
[PythonAnywhere](https://www.pythonanywhere.com/).

Detailed instructions for developing this app are available
[here](https://blog.memcachier.com/2018/10/15/django-on-pythonanywhere-tutorial/).
